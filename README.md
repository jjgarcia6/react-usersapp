# react-usersapp

Proyecto ejemplo en React (Vite). El subproyecto principal está en `6-users-app`.

## Requisitos

- Node.js (recomendado: 18+)
- npm (o pnpm/yarn)

## Quick start (desde la raíz)

1. Entrar al subproyecto:

   ```bash
   cd 6-users-app
   ```

1. Instalar dependencias:

   ```bash
   npm install
   ```

1. Ejecutar servidor de desarrollo:

   ```bash
   npm run dev
   ```

1. Abrir la aplicación en el navegador (por defecto):

   ```bash
   $BROWSER http://localhost:5173
   ```

## Estructura básica

- `index.html` - HTML de entrada
- `src/main.jsx` - punto de arranque
- `src/UsersApp.jsx` - componente raíz

Más documentación y archivos relacionados se encuentran dentro de `6-users-app/`.
